# 60-30-10 Color Usage: Moroccan Luxury

This project follows a 60-30-10 scheme tailored for the high-end Moroccan fashion market, emphasizing heritage deeps and metallic sparks.

- **60% Dominant (Foundation):** `aq-ink` (#0F172A). Deep blue-black providing a luxurious, focused workspace. Used for main backgrounds and page wrappers.
- **30% Secondary (Character):** `aq-indigo` (#3F57AF). A rich, heritage-inspired indigo representing the Moroccan soul. Used for large section blocks, secondary surfaces, and card accents.
- **10% Accent (Spark):** `aq-gold` (#D4AF37). A burnished gold mimicking artisanal metalwork. Reserved for primary CTAs, active states, and critical highlights.

## Theme Tokens

Defined in `app/globals.css` as CSS variables mapped to Tailwind tokens:

- **Dominant:** `--tone-dominant` → `bg-background` / `text-foreground`
- **Secondary:** `--tone-secondary` → `bg-secondary` / `text-secondary-foreground`
- **Accent:** `--tone-accent` → `bg-primary` / `text-primary-foreground`

Utilities are provided for convenience:

- `bg-dominant`, `text-dominant`
- `bg-secondary-surface`, `text-secondary-surface`
- `bg-accent-surface`, `text-accent-surface`

## Practical Guidance

- **Accessibility:** Ensure high contrast for text. Our `primary-foreground` is mapped to `aq-ink` when used on `primary` (Gold) backgrounds to maintain readability.
- **Motion:** Use `aq-blue` and `aq-purple` gradients for kinetic elements, but anchor the static UI in the 60-30-10 rule.
- **Low-End Support:** Avoid complex transparency layers on low-end devices; use solid `bg-secondary` instead of `glass`.
