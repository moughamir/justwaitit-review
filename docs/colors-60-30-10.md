# 60-30-10 Color Usage

This project follows a 60-30-10 scheme:

- 60% Dominant: neutral backgrounds and the main canvas.
- 30% Secondary: supporting surfaces and large section blocks.
- 10% Accent: CTAs, highlights, and small UI elements.

## Theme Tokens

Defined in `app/globals.css` as CSS variables mapped to Tailwind tokens:

- Dominant: `--tone-dominant` → `bg-background` / `text-foreground`
- Secondary: `--tone-secondary` → `bg-secondary` / `text-secondary-foreground`
- Accent: `--tone-accent` → `bg-primary` / `text-primary-foreground`

Utilities are provided for convenience:

- `bg-dominant`, `text-dominant`
- `bg-secondary-surface`, `text-secondary-surface`
- `bg-accent-surface`, `text-accent-surface`

## Practical Guidance

- Pages: keep the page wrapper on the dominant background (`bg-background`).
- Sections: use one major section per view with `bg-secondary` to represent ~30% coverage. Example: Lookbook section updated to `bg-secondary`.
- Accent: reserve `bg-primary` or `bg-brand-diag` for CTAs, badges, and small highlights only.

If you prefer purple as the accent, switch `--tone-accent` to `--accent` in `globals.css`.
