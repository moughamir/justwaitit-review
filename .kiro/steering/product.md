# Product Overview

Anaqio is an AI-driven virtual studio for fashion commerce that generates studio-quality fashion imagery instantly. The platform empowers fashion brands with cutting-edge digital tools for modern commerce.

## Current Status

Pre-launch waitlist phase with early access program.

## Core Features

- AI-powered fashion photography generation
- Virtual studio capabilities
- Waitlist and early access management
- User authentication and protected areas
- Brand identity showcase

## Target Audience

Fashion brands seeking automated, high-quality visual content for e-commerce and marketing.

## Brand Identity

- Primary colors: Blue (#2563EB), Purple (#7C3AED)
- Gradient palette: #3F57AF → #484DA9 → #6049A8 → #6F47A7
- Typography: Space Grotesk (display), Inter (UI/body), Instrument Serif (editorial)
- Tone: Professional, cutting-edge, empowering
